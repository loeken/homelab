    Download Clients¶

    Settings -> Download Clients

    rtorrent-flood¶
    • select option flood
    • Host: rtorrent-flood
    • UrlBase: /
    • Username/Password ( as you set in ../rtorrent-flood/README.md )
    nzbget¶
    • select option nzbget
    • Host: nzbget.media
    • Category: MOVIES
    • username ( what you configured in nzbget )

    • password ( what you configured in nzgbet )

    • test

    • save
